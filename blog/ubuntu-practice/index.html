Ubuntu 安装、设置与使用

Homecat

Ubuntu 是一个以桌面应用为主的 Linux 操作系统，因其开源软件之属性而获得广泛应用。

Ubuntu 20.04 LTS 的安装

下载安装文件

在 Ubuntu 官网 ubuntu.com，可以下载到 Ubuntu 20.04 LTS 的安装文件。安装文件有多种版本，也可以采用不同的安装方式。客户端可以使用的最新的长期支持的稳定版本为：ubuntu-20.04.2.0-desktop-amd64.iso。 如果主站的下载速度比较慢，可以在镜像网站选择速度快的。

制作安装盘

使用 Rufus 制作 Ubuntu 系统的安装盘。由于 Rufus 只能在 Windows 上运行，并且要求 Win7 及后续版本(32或64位均可)，因此制作过程要在 Windows 环境中进行。

  • 从 Rufus 官网 rufus.ie下载 rufus.exe 文件。
  • 制作过程简单，在程序界面上依次选择下载的 ISO 文件、启动模式等参数后，按开始即可。（特别说明：启动模式的选择很重要，和要安装电脑的启动模式有关系，BIOS 或者 UEFI，必须正确选择。）

单系统安装步骤

  • 将USB安装盘插入待安装的电脑。
  • 打开电脑，进入 setup，选择从 USB 启动，进入 Ubuntu 安装程序，按部就班就可以了。

Uget 的安装和设置

Uget 是一款优秀的下载管理工具，可以提高下载速度。Uget 可以作为独立的应用安装，也可以作为扩展安装到浏览器上，本例是以 Firefox for Ubuntu 为对象，安装 Uget 扩展。

安装 uget 和 aria2

sudo apt install uget
sudo apt install aria2

配置 Uget

打开安装好的 Gget，在菜单上依次选择 Edit > Settings > Plug-in > Plug-in matching order > aria2，将 aria2 配置为优先。

至此，Uget 已经可以独立使用了，为了在浏览器上也可以使用，需要将 Uget 安装为扩展。

安装 uget-integrator

sudo add-apt-repository ppa:uget-team/ppa
sudo apt update
sudo apt install uget-integrator

安装扩展

打开 Firefox 浏览器，找到并点击右上角 Menu，选择 Add-ons；在 Add-ons 的界面的 Find more add-ons 中搜索 uget；在结果列表中选择 uGet Integration；点击 Add to Firefox 按钮即可。

常用命令

安装和删除程序

sudo apt install <程序名>
sudo apt remove <程序名> //删除应用（保留配置文件）
sudo apt --purge remove <程序名> //删除应用(不保留配置文件)

加载和卸载软件包

sudo add-apt-repository ppa:<软件包名> //家在软件包
sudo add-apt-repository --remove ppa:<软件包名> //卸载附加软件包
sudo apt update //更新软件包
sudo apt autoremove //卸载多余的软件包
