Git 的安装和使用

Homecat

Git 是一个开源的分布式版本控制系统，也是内容管理系统(CMS)和工作管理系统，适用于各类项目。

安装 Git

在 Ubuntu 20.04 LTS 系统上安装 Git 的最新版本

sudo add-apt-repository ppa:git-core/ppa
sudo apt update
sudo apt install git

Git 命令

Git 的使用方法请访问 Git官网。

配置账号

git config –global user.name <your_name>     //配置用户名
git config –global user.email <your_email>     //配置电子邮箱
git config -l     //查看配置信息

新建仓库

mkdir orchid
cd orchid
git init
git remote add origin https://gitee.com/homecat805/orchid.git
git add -A
git commit -m "#####"
git push -u origin master

已有仓库

git push origin master
