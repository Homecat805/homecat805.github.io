Hugo-静态网站生成器

Homecat

Hugo 是由 Go 语言实现的静态网站生成器，简单、易用、高效、易扩展、快速部署。

Hugo 安装

下载及安装

在 github.com 可下载各种版本。版本列表

适用 Ubuntu 20.04 LTS 系统的安装文件以 .deb 为扩展名；如 Hugo 主题使用 SASS 预处理语言，则需要选择包含 extended 字样的安装文件。下载 .deb 文件后，可直接点击安装。

其他安装方法

wget https://github.com/gohugoio/hugo/releases/download/v0.83.0/hugo_extended_0.83.0_Linux-64bit.deb
sudo dpkg -i hugo_0.75.1_Linux-64bit.deb

检查安装结果

使用 hugo –help 命令，出现说明信息，表示安装成功；

hugo --help
hugo is the main command, used to build your Hugo site.
(以下省略)

也可以使用 hugo version 命令查看版本。

hugo version
Hugo Static Site Generator v0.73.0-428907CC linux/amd64 BuildDate: 2020-06-23T16:30:43Z

Hugo 使用

创建网站

hugo new site <静态网站名称>

安装主题

Hugo官网提供了许多主题，有两种安装方法：一是将主题文件下载下来，放置到 themes 目录下；另一种是直接用 git clong 命令。

$ git clone <远程主题地址> themes/<本地主题名称>

或自建主题。

hugo new theme <主题名称>

运行网站

运行网站前，都需要编辑配置文件，每个主题的配置方法，大致相同，也会略有差别，具体的配置方法参看说明文件。一般情况下，主题里会有一个 exampleSite 目录，内有 config.toml 文件，根据该文件的内容，修改根目录的 config.toml 文件。

hugo server

运行 hugo server 后，可通过 http://localhost:1313 访问网站，但这时的网站还不是静态网站 。

生成静态网站

hugo

运行 hugo 命令，会产生 public 目录，目录下就是静态网站的文件，将这些文件上传到网站空间，就可以访问了；将静态网站部署到 Github 或者 Gitee 都是不错的选择。
